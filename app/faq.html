Frequently Asked Questions

Everything you need to know about navigating the application interfaces smoothly.

How do I use the typing Transliteration engine? +
Click on the globe icon (🌐) next to the conversation input stream, choose your target native language, and type phonetically using your English keyboard. Pressing the **Spacebar** will instantly convert the phonetics to script characters.
How can I view message-seen metrics in community pools? +
For any group context message you broadcast, an active interactive timestamp flag labeled "✓ Checked" appears underneath. Click directly on that token check link to spin up a granular dynamic tracking audit modal revealing who saw it and when.
What file transfer limitations are configured on disk? +
You can upload images, micro-videos, audio records, or structural PDF data documents. The platform limits file processing batches to a maximum of 10 media elements per upload operation with an explicit size constraint ceiling of 15MB.
